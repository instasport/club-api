# Интерфейс системы Instasport (API)

Система **Instasport** имеет интерфейс, который позволяет подключаться к ней с сайта клуба или из мобильного приложения.

Для получения информации через **API** (например: узнать расписание тренировок или получить список абонементов) или регистрации/авторизации клиента необходимо иметь ключ доступа (**API Key**).

Для совершения каких-либо действий от имени клиента (например: записаться на тренировку или купить абонемент) нужно авторизоваться в системе и получить токен доступа (**access token**).  

Описание находится на [Wiki](https://github.com/instasport/club-api/wiki)
